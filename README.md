This is a personal website built with [Vite](https://vite.dev), React, TypeScript, and Tailwind CSS v4.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
```

```bash
npm run dev
```

Open the URL printed by Vite (usually [http://localhost:5173](http://localhost:5173)).

Main entry points:

- `src/main.tsx`
- `src/App.tsx`
- `src/components/*`
- `src/styles.css`

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
